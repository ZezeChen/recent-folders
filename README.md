
# 最近文件夹

快速访问最近打开的文件夹。

- 使用方法
    1. 点击侧边栏的资源管理器图标，将侧边栏的"最近文件夹"图标拖动到资源管理器视图中
    2. 点击文件夹即可打开该文件夹
- 配置
    - `recentFolders.maxItems`: 文件夹显示数量（默认：20，范围：1-50）

# Recent Folders

Quick access to recently opened folders.

- Usage
    1. Click the Explorer icon in the sidebar, then drag the "Recent Folders" icon into the Explorer view
    2. Click a folder to open it
- Configuration
    - `recentFolders.maxItems`: Number of folders to display (default: 20, range: 1-50)
